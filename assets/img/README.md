# Site images

The sandbox this site was built in can't reach coachingbyrami.com or Canva, so
photos must be exported manually and dropped into this folder. The site works
without them (missing images hide themselves), and picks them up as soon as
they exist with these exact names:

| File | Used where | Suggested source |
|------|-----------|------------------|
| `raminta.jpg` | About section portrait | Old site headshot or Canva brand kit |
| `hero.jpg` | (optional, future) hero background | Canva design |

How to export:

- **Old site:** open https://coachingbyrami.com/, right-click each photo →
  "Save image as…".
- **Canva:** open the brand design → Share → Download → PNG/JPG, then crop as
  needed.

Keep images under ~300 KB each (use https://squoosh.app to compress) so the
page stays fast.
